# 🚀 БЫСТРЫЙ СТАРТ - 5 МИНУТ

## 1️⃣ Откройте PowerShell

Нажмите `Win+R` и введите `powershell`, потом Enter.

## 2️⃣ Перейдите в папку проекта

Скопируйте и вставьте:

```powershell
Set-Location "c:\Users\Andrey\Documents\GitHub\magazine_site\admin_panel"
```

## 3️⃣ Запустите сервер

```powershell
python manage.py runserver
```

Должно появиться:
```
Starting development server at http://127.0.0.1:8000/
```

## 4️⃣ Откройте браузер

Перейдите на: **http://localhost:8000**

Вы должны увидеть сайт Рыбный День со всеми товарами! 🎣

## 5️⃣ Протестируйте

1. Нажмите на любой товар
2. Выберите вес или количество
3. Нажмите "Добавить в корзину"
4. Нажмите иконку корзины (сверху справа)
5. Заполните форму (важно: Имя и Телефон)
6. Нажмите "Оформить заказ"

Если видите зеленое сообщение "✅ Заказ успешно оформлен!" - **все работает!**

---

## ⚙️ Настройка Email (Опционально)

Если хотите чтобы приходили письма при заказе:

1. Откройте файл: `admin_panel/fish_day_admin/settings.py`
2. Найдите строку `EMAIL_HOST_PASSWORD = 'example-app-password'`
3. Получите пароль приложения Яндекс (процесс описан в `COMPLETE_SETUP_GUIDE.md`)
4. Замените `'example-app-password'` на реальный пароль
5. Перезагрузите сервер (Ctrl+C и запустите заново)
